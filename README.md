## ![doom rage4](https://github.githubassets.com/images/icons/emoji/rage4.png) choix de la langue :

Le jeu prend vos parametres de language de Steam.

Si ce n'est pas le cas vous pouvez les changer en suivant les étapes suivantes :

1 - aller dans le dossier d'installation du jeu ( dans ce cas : ` E:\Games\Cracked\Dying Light 2 `)

2 - naviguer jusqu'à ` steam_settings `

3 - ouvrir ` force_language.txt `

4 - changer la langue ` french `

5 - naviguer jusqu'à ` CTRL + S ` ou ` fichier -> enregistrer `

6 - fermer la fenetre

7 - naviguer jusqu'à ` Dying Light 2\engine\source\bin\x64 `

8 - ouvrir ` tenoke.ini `

9 - à la ligne : ` language = "english" ` 

remplacer ` english ` par la langue souhaité parmis celles citées au dessus de cette même ligne.

10 - à la ligne : ` country = "UK" ` 

remplacer ` UK `. pour trouver votre code de pays ISO 3166-1-alpha-2 voir à cette adresse :

` https://www.iban.com/country-codes `

11 - ` enjoy !!! `
